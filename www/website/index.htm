What is JForum?

JForum is a powerful and robust discussion board system implemented in Javatm. It provides an attractive interface, an efficient forum engine, an easy to use administrative panel, and an advanced permission control system.

Built from the ground up around a MVC framework, and implemented as a Javatm servlet, it can be deployed on any servlet container such as Tomcat or jBoss, and its clean design and implementation make JForum easy to customize and extend.

Best of all, JForum is freely available under a BSD license.
