What is JForum?

JForum is an extremely powerful and fast discussion board system, built using the Javatm Technology.
Aside many other features, it provides a fully customizable interface, really fast and robust forum engine, Friendly URLs, easy to use Administrative Panel, unlimited forums and users, along with a Permission Control System that allow fine tunning of the forum's settings.

If you or your company are searching for a serious Forum product, JForum is the right choice.


Key Features

 Multi-language support

 Customizable user-interface

 Secure and higly customizable Permission Control

 Multi database support

 Unlimited foruns and topics
