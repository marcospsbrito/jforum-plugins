Features

Below are listed some of the features present in JForum.

Easy migration from PHPBB

Support for multiple databases

Easy and fast development for any other data source

Unlimited forums, categories and topics

Higly customized permission control. It is possible to control access to any forum and category, per group and per user, including a lot of fine tunning options

Optimized and fast seach engine

Unlimited users

Unlimited groups

Unlimited groups per user

User profile

Message posting allows many different text formatting choices

BB Tags support

Easy to support new bbtags via a XML configuration file

Email notification when a topic is answered

Private Messages

Email notification on new Private Message

Unlimited emoticons, easly administrable via a configuration panel

Easy to moderate. It's possible to create "Forum Moderators", that are users with special privileges to ensure the forum's quality

Moderators may edit, remove and lock posts or entire topics quickly and easily

Different levels of moderation, being possible to give some users limited moderation options and, to other users, full moderation options

Easy to customize the forum layout

Unlimited templates / layouts per forum

Each user can each it's own forum template. The administrator set the default forum look'n feel, and then let users customize the layout via a pre-defined set of templates

Listing of currently online users

User level identification by colors. For example, green for Administradors, orange for Moderators and so on

User Ranking - each user receive different ranking levels depending the number of the messages posted by him / she

Fast! Fast! Fast! JForum does a lot of caching to ensure performance and quick feedback for users

Support to RSS

Friendly URLs. Forum has friendly and easy to remeber URLs for anything

Forums and topics have different icon images depending of hits status ( locked, unlocked, forum / message read / unread

Possibility to lock / unlock topics

Possibility to watch a topic wihout being necessaary to post any message to it

Support different languages

Easy to add new translations to the forum

Users may choose the preferred board language given a set of available translations

Forum remembers the date and time of the last visit of each user, and can display a list of unread topics and forums since its last visit

Good looking layout

Secure. JForum has a robust security system

Topic moving. It is possible to move topics between forums

Easy to administer board. It is possible to set all board properties via a easy-to-use web interface

Avatars. Users may uplaod avatars or specify an external image, which will be associated to its user

User blocking. With a single click, board administrators may block / unblock user-accounts

Possibility to only allow registration of new users after a confirmation sent to user's email, improving security and not allowing robots to register

Topics can be set as "Sticky" and "Announce", that are special topics which are always shown first

Post Preview. Before posting the message, users may use a preview option to see how the message will look like

Advanced HTML filter
