Help

What database does JForum support?
As of Release 5, JForum supports MySQL, PostgreSQL and HSQLDB.

Which are the languages currently supported in JForum?
As of Release 5, the languages supported are English, Netherlands, Português do Brasil, French and German.

If I make a post "Sticky", what does that mean? Also, what does "Announce" mean, other than a special icon?
Announces are always shown first. Sticky are shown first of other topics as well, but after Announces.

What framework is used for the development of JForum?
JForum uses Freemarker ( http://www.freemarker.org ) as template engine. There are no other major dependencies or frameworks.

Which open source license is JForum under?
JForum is licensed under the BSD License

Is it possible to completely turn off email notifications?
Yes, you can. You can change this option in the admin panel, Configurations section.
