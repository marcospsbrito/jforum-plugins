    How do I configure the Forum settings?
    All administrative settings are done in the "Admin Panel", that can be accessed through the link "Admin Panel" available in the end of the page. You must be logged as administrator, either by using the Admin account or any other user who has administrative privileges.

    Which are the languages currently supported in JForum?
    As of version 2.0, the languages supported are English, Chinese Simplified, French, German, Dutch, Português do Brasil, and Russian.

    What does "Announce" and "Sticky" mean, other than a special icon?
    Announces are always shown first. Sticky are shown first of other topics as well, but after Announces.

    Which open source license is JForum under?
    JForum is licensed under the BSD License

    How do I configure email notifications?
    You can change this option in the admin panel, Configurations section.

    Avatar uploading is not working. What's wrong?
    If you are using Linux, or other Unix like machine, it is necessary that you have installed the X server libraries. Another solution is to run the Servlet Container setting up JAVA_OPTS's parameter java.awt.headless to "true". For more information, see http://www.jforum.net/posts/list/219.page

    How to I set a different Character Encoding?
    Go to the Admin Panel, section "Configurations". Set the character encoding you want in the "Character Encoding" field.
